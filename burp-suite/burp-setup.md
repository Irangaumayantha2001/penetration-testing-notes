## Objective
Configure Burp Suite Community Edition for intercepting HTTP traffic.

## Tools
- Burp Suite Community
- Firefox Browser

## Configuration Steps
1. Launch Burp Suite
2. Open Firefox proxy settings
3. Set proxy to:
   - IP: 127.0.0.1
   - Port: 8080

## Certificate Installation
- Visit http://burp
- Download CA certificate
- Import certificate into Firefox

## Key Burp Modules Used
- Proxy
- Repeater
- Intruder (limited in Community)

## Purpose
Burp Suite is used to analyze and modify HTTP requests to identify security vulnerabilities.
